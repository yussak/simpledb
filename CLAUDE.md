## プロジェクト概要

学習目的で RDBMS をゼロから自作するプロジェクト。
外部のデータベースライブラリは使わず、ストレージ層からクエリ実行までを Go でスクラッチ実装する。

## 頻用コマンド

```sh
go run .           # 実行
go build ./...     # ビルド
go test ./...      # テスト
go fmt ./...       # フォーマット
go vet ./...       # 静的解析
```

Docker で実行する場合:

```sh
docker build -t simpledb .
docker run --rm simpledb
```

## コーディング規約

- Go の標準スタイルに従う（`gofmt` / `go vet` を通すこと）。
- コメントは日本語で書く。
- エラーは握りつぶさず、適切に呼び出し元へ返す。

## 設計方針

- 外部の DB ライブラリには依存せず、自前で実装する。
- 学習目的なので、性能よりも分かりやすさ・読みやすさを優先する。
- 一度に大きく作らず、小さい単位で実装・検証を繰り返す。
