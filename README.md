# simpledb

学習目的で RDBMS をゼロから自作するプロジェクト。
外部のデータベースライブラリには頼らず、ストレージ層からクエリ実行までを Go で実装していく。

## ステータス

ごく初期段階。現状は雛形（hello world）のみで、これから各コンポーネントを実装していく。

## 技術スタック

- 言語: Go 1.23
- 実行環境: Docker（任意）

## 開発ロードマップ

「動くものを最短で作り、そこに肉付けする」方針で、依存関係が素直になる順に実装していく。

| Phase | 内容 | ゴール / 検証 |
|---|---|---|
| 0 | 字句解析・構文解析（`CREATE` / `INSERT` / `SELECT` のみ）＋ メモリ上テーブル ＋ 素朴な実行器 | SQL を打つと結果が返る。`INSERT` 後に `SELECT` で取れることをテストで確認 |
| 1 | `WHERE`（フルスキャン＋フィルタ） | 条件付き検索が返ること |
| 2 | ファイル永続化（ページ単位のストレージ） | 再起動してもデータが残ること |
| 3 | B+木インデックス ＋ 主キー検索 | フルスキャンとインデックス検索で同じ結果が返ることを照合 |
| 4 | WAL ＋ トランザクション | 書き込み途中のクラッシュを WAL から復旧 |
| 5 | TCP 接続（簡易な独自プロトコル） | ネットワーク経由でクエリを実行できること |

### 設計メモ（早めに固定する前提）

- 型は MVP では `INT` と `TEXT` の 2 つに絞る。
- ページサイズは 4KB 固定。
- AST / 内部値表現は Phase 0 で決める（全 Phase に波及するため丁寧に）。
